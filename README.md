# Secure WordPress Deployment (Linux + Nginx + MariaDB + PHP)

## Overview
This project demonstrates a complete secure WordPress hosting setup on RHEL Linux.
It includes Nginx, PHP-FPM, MariaDB, SFTP user isolation, phpMyAdmin security,
SSL (HTTPS), and WordPress CMS configuration.

## Features
- Nginx + PHP 8.x + MariaDB 10.x
- WordPress deployed under /home/wordpress/websitename/public
- Secure SFTP access (chrooted user)
- phpMyAdmin secured with HTTP Basic Auth
- HTTPS enabled using free SSL
- Full troubleshooting documentation

## Tech Stack
- OS: RHEL / Amazon Linux
- Web Server: Nginx
- Database: MariaDB
- Language: PHP
- CMS: WordPress
- Automation Structure: Ansible-ready
